New Features in 5.6.0
---------------------

*   LevelDB Store
*   MQTT transport
*   New LDAP security module
*   Stomp 1.1 support
*   stomp+nio+ssl transport
*   Multi KahaDB persistence
*   Priority Failover URIs
*   Automatic client rebalance in broker cluster
